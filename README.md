## Getting Ruby Set up and necessary gems 

* You would install the latest version of the gems listed in the Gemfile by :

```bash
$ bundle install
```
* Migrate the database with by :
```bash
$ rails db:migrate
```
* Simply run the app with :
```bash
$ rails s
```
